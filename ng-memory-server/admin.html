Integration Guide
Point any OpenAI-compatible chat UI here
Any chat interface that supports a custom base URL works with NodeGhost out of the box. Just set these values:
Base URL: https://nodeghost.ai/v1 API Key: ng-your-key Model: deepseek-chat // or your own model
Query your knowledge base (no auth required)
Use the public recall endpoint so customer-facing agents can query your knowledge base without exposing your ng- key:
POST http://your-server:3100/public/recall { "query": "what is your refund policy?", "namespace": "knowledge-base" } // Headers: X-Owner-Token: ng-your-key // never exposed to customers

Open WebUI — Self-hosted, full featured LibreChat — Open source ChatGPT alternative Chatbot UI — Clean minimal interface Jan — Desktop app, local + cloud models AnythingLLM — Business focused, team features
